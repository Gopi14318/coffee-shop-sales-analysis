# Coffee Shop Sales Analysis - Data Analyst Project

## Objective
Analyze coffee shop transaction data to understand revenue, product performance, store performance, peak sales time, and payment trends.

## Files Included
1. coffee_shop_sales_data.csv - raw sample dataset
2. coffee_shop_data_analyst_project.xlsx - Excel project with dashboard, raw data, SQL queries, and insights
3. coffee_shop_sql_queries.sql - SQL queries for analysis

## Tools Used
- Excel: Data cleaning, dashboard, charts, KPI cards
- SQL: Business analysis queries
- Data Analysis Concepts: KPI tracking, aggregation, trend analysis, product/store analysis

## Dashboard KPIs
- Total Revenue
- Total Orders
- Total Quantity Sold
- Average Order Value
- Best Selling Product

## Key Analysis
- Monthly revenue trend
- Top 10 selling products
- Revenue by category
- Store-wise revenue
- Peak hour sales
- Weekday sales
- Payment method split

## Resume Points
Coffee Shop Sales Analysis Dashboard
- Analyzed 900 coffee shop transactions using Excel and SQL.
- Built an interactive Excel dashboard with KPIs, trend charts, and product/store performance summaries.
- Identified best-selling products, peak sales hours, and top revenue-generating stores.
- Wrote SQL queries to calculate revenue, order volume, category performance, and monthly trends.

## Interview Explanation
This project analyzes coffee shop sales data. I cleaned and structured transaction-level data, calculated KPIs like total revenue, total orders, quantity sold, and average order value. I used SQL for querying insights and Excel for dashboard creation. The dashboard helps business users understand which products sell the most, which store performs best, and what time sales are highest.
